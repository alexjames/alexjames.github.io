## Arrays
Arrays are a collection of elements stored in contiguous memory locations.

```c++
// Declared on the stack
int arr[10];

// Declared on the heap
int *arr = new int[10];
```

```java
// Declared on the heap, you cannot store arrays on the stack in Java
int[] arr = new int[10];
```
